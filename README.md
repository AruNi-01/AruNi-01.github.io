# About

- 一款配置简单，部署容易的博客。
- 通过 [GitHub Pages](https://docs.github.com/cn/pages/quickstart) & [GitHub Actions](https://docs.github.com/cn/actions)（Github 提供的 `CI/CD` 服务） 部署。

## Usage

> 需要 node.js 环境

1. 克隆此仓库到本地，下载依赖：
   > 若未使用过 `pnpm` 包管理器（高性能的 npm），请先下载：`npm i -g pnpm`

   接着继续执行：

    ```bash
    pnpm i
    ```

2. 在本地启动项目:

    ```bash
    pnpm docs:dev
    ```
   
3. 访问 `localhost:8080`
